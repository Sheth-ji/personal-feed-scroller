# Personal Feed Scroller

A privacy-focused, local-first web app that lets you scroll through your personal media collection in a beautiful Instagram-like feed - without the ads, algorithms, or tracking.

## Why This Exists

Tired of social media algorithms? Miss the days when software just did its job without manipulating you? This tool lets you enjoy the satisfying experience of scrolling through content, but with *your* photos, videos, and memes - all stored locally on your device.

## How to Use

**Option 1: Use Directly (Easiest)**
1. **Visit** https://ignatius-42.github.io/personal-feed-scroller/
2. **Click "Choose Folder"** and select a folder with your images/videos
3. **Start scrolling!** The app will shuffle and display everything

**Option 2: Download and Use Offline**
1. **Download** the `index.html` file from this repository
2. **Open it** in any modern browser (Chrome, Firefox, Safari, Edge)
3. **Click "Choose Folder"** and select a folder with your images/videos
4. **Start scrolling!** Works completely offline

That's it. No installation, no configuration, no signup.

## Security

**Is my data safe?**

Yes. Here's how:

ğŸ”’ **Browser Isolation** - Your photos are isolated from other websites by your browser's built-in security (Same-Origin Policy)

ğŸ”’ **No Network Access** - The app doesn't connect to the internet at all. Even if someone compromised another website you're visiting, they cannot access your photos through this app.

ğŸ”’ **HTTPS Delivery** - When hosted on GitHub Pages, the app is served over HTTPS, preventing tampering during download

ğŸ”’ **Open Source** - All code is visible. You (or anyone technical) can inspect it to verify there's no malicious behavior

ğŸ”’ **No Persistence** - When you close the tab, all loaded data is cleared from memory

**The only real risk:** Malicious browser extensions that have access to all your tabs. This is outside any web app's control - choose your browser extensions wisely!
